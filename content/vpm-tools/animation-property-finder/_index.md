---
title: Animation Property Finder
---

Quickly find what animation clips animate a given property on a GameObject.

![](/images/vpm-tools/animation-property-finder/window.png "Main window")

# How to use

1. Drag in a GameObject into the `Object` field
2. Either select a preset, or copy a property name *(right-click in a poiyomi material)*
3. Search, this will search in all animations in your project, so even if they are not in an animator

> [!TIP]
> Use `*` as a wildcard, this means any character or number can be there, and can occur 0-infinite times.
